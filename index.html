Ludopoiesis

Un oráculo para jugar la vida.

Un espacio para detenerte, hacer una pregunta
y mirar el mundo desde otra perspectiva.

No busca respuestas definitivas.
Invita a sintonizar, asombrarte,
crear y fluir con el juego.
